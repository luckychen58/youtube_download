## ✨ 核心特点

- **真正无后台**：运行后无任何托盘图标、无服务、无开机自启
- **单文件绿色版**：解压即用，无需安装，体积小巧
- **支持超高清**：4K、8K、HDR（视原视频支持情况）
- **批量下载**：支持播放列表、频道（部分实现）
- **格式灵活**：可选择视频+音频合并（mp4/mkv）、纯视频、纯音频（mp3/m4a）
- **智能命名**：自动按标题+上传日期+分辨率命名，方便整理
- **代理支持**：内置 socks5/http 代理设置（适合科学上网环境）
- **无任何联网上报**：不收集使用数据、不打电话回家

## ⚡ 下载 & 使用

### 最新版本 v1.0（2026年发布）

- **GitHub Releases**（推荐）：[前往 Releases 下载 →](https://github.com/luckychen58)  
- **备用网盘**（国内加速）：蓝奏云 / 123盘 / 夸克 等（自行上传后替换链接）

**文件名称示例**：`YouTube4k无后台-下载器-v1.0.exe`  
**大小**：约 80–120 MB（包含必要的运行库）

### 快速上手（3秒启动）

1. 双击 exe 文件运行（**无需管理员权限**）
2. 粘贴 YouTube 视频/播放列表链接
3. 选择画质 & 格式（默认推荐最高可用 mp4）
4. 点击“开始下载” → 等待完成
5. 下载路径默认为：**exe 所在目录 → Downloads** 文件夹
